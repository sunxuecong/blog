VSCode下载

推荐一些在开发中常用的插件

  • Auto Close Tag：自动闭合标签
  • Auto Rename Tag：自动重命名标签
  • Code Translate：滑词翻译插件
  • colorize：可视化颜色
  • CSS Variable Autocomplete：CSS变量自动补全
  • DotENV：.env文件支持高亮
  • EditorConfig for VS Code：使用.editorconfig支持编辑器格式配置
  • Error Lens：直接在编辑器里错误提示
  • ESLint：ESlint支持插件
  • Git Graph：Git分支查看
  • Git History：查看文件Git历史
  • git-commit-plugin：Git提交信息插件
  • GitLens：编辑器中快捷查看git提交信息
  • i18n Ally：i18n翻译便利插件
  • Iconify IntelliSense：编辑器可视化图标
  • Image preview：快捷查看导入的图片素材
  • Smart Clicks：编辑器点击选取增强插件
  • Template String Converter：普通字符串自动转模板字符串
  • Turbo Console Log：控制台快捷输出
  • UnoCSS：UnoCSS支持插件
  • Vue - Official：Vue官方支持插件
